# Overview 
The Report Catalog is a python based Flask program that collects report information pertaining to multiple platforms across our used programs within our ecosystem. This catalog collects all pertaining report data and is then displayed in the program by each report gathered.
there is a job that is ran everyday at gets the most updated information and stores it within the catalog. This job is ran at 10PM.
